## 🔄 Content Lifecycle

### Stage 1: Planning
```
1. Review content calendar
2. Identify topic from category
3. Align with current theme
4. Assign content framework
5. Set target keywords
```

### Stage 2: Creation
```
1. Research topic thoroughly
2. Outline content structure
3. Write in Markdown format
4. Convert to HTML format
5. Source Unsplash image
```

### Stage 3: Optimization
```
1. SEO optimization check
2. Mobile responsiveness test
3. Brand compliance review
4. Quality checklist pass
```

### Stage 4: Publication
```
1. Upload to platform
2. Schedule publication
3. Update content calendar
4. Track performance metrics
```

### Stage 5: Maintenance
```
1. Monitor engagement
2. Update outdated information
3. Refresh evergreen content
4. Archive obsolete content
```

## 🔍 SEO Strategy

### Keyword Approach

**Primary Keywords (High Volume):**
- AI tools for business
- Business productivity tips
- Automation for small business
- Business growth strategies

**Long-tail Keywords (High Intent):**
- Best AI tools for small business 2026
- How to automate business processes
- Productivity tools for remote teams
- Business mindset for entrepreneurs

### On-Page SEO Requirements
- [ ] Keyword in title (preferably beginning)
- [ ] Keyword in first 100 words
- [ ] Keyword in H2 headings (1-2 times)
- [ ] Meta description with keyword
- [ ] Alt text with relevant keywords
- [ ] Internal linking to related content
- [ ] External links to authoritative sources

### Content Length Guidelines
| Post Type | Target Length |
|-----------|---------------|
| Quick Tips | 500-800 words |
| Standard Posts | 800-1,200 words |
| Deep Dives | 1,500-2,500 words |
| Ultimate Guides | 3,000+ words |

## 🔄 Evergreen Content Strategy

### Principles
1. **Avoid temporal references** - No "this year" or specific dates unless necessary
2. **Focus on timeless principles** - Core concepts that remain relevant
3. **Regular updates** - Review and refresh quarterly
4. **Modular structure** - Easy to update sections independently

### Evergreen Categories
- Business Mindset principles
- Strategic frameworks
- Foundational concepts
- Best practices (timeless)

### Time-Sensitive Categories
- Industry Updates (date accordingly)
- Trends (review quarterly)
- Tool reviews (update regularly)
- News-related content

### Minimum Requirements

- [ ] Relevant image from Unsplash

- [ ] Expert quotes (cited)
